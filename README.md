# wawjs-snippets

School pet project.

Distributing and installing Snippets for:

- JavaScript
	- node.js APIs
	- mocha
	- ...
- for editors
	- Atom
	- ...

## Installing

	npm install https://github.com/trogper/wawjs-snippets.git

This will create

	$HOME/$ATOM/snippets.cson snippets

## Development

	https://github.com/trogper/wawjs-snippets.git
